# dot files

use to backup my dot files.

## dot files to back up

- .zshrc
- .bashrc
- .zshenv
- .gitconfig
- .gitignore
- .m2/settings.xml
- .pip/pip.conf
- .ssh/config
- .ssh/known_hosts
- tmux.conf
- .vimrc
- .config/nvim/init.vim
- .config/clash/config.yaml

## 本质

- file: file to backup
- name: config file name
- group: can be null

link file to original file. using git or rsync , to gzip to backup them.
